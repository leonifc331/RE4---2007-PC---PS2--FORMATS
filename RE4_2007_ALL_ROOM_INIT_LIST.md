# RE4 2007 — Lista completa de INIT por sala

Arquivo base analisado: `BIO4.exe` / `RE4 2007 PC`.

Cada sala normalmente possui dois ponteiros INIT: `INIT A` e `INIT B`. Para fazer INIT Swap, copie os bytes LE da sala de origem para os offsets de patch da sala alvo.

Campos importantes:

- `INIT_A_VA` / `INIT_B_VA`: endereço virtual da função INIT.
- `INIT_A_Bytes_LE` / `INIT_B_Bytes_LE`: bytes little-endian para colar no HxD.
- `Patch_A_FileOffset` / `Patch_B_FileOffset`: offset no EXE onde o ponteiro deve ser trocado.
- `SafeDirectSwap`: `YES` = pode trocar direto; `NO_SHARED_OR_INCOMPLETE` = cuidado, ponteiro compartilhado via registrador.
